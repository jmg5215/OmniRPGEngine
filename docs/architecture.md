# OmniRPGEngine Architecture

## High-Level Components
- UI_Main: Main panel wrapper, navigation, and top-level layout
- UI_Leaderboard: Leaderboard page layout and actions
- UI_Rage: Rage tree layout and interactions
- Systems_XP: XP gain, level thresholds, integration with XPerience plugin
- Systems_Rage: Internal Rage logic, buffs, cooldown handling
- Systems_Leaderboard: Data collection and ranking logic
- Utils: Common helpers and extensions
